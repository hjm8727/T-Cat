###  프로젝트 주요 기능
- 회원이 명시적으로 작품 정보를 확인할 수 있도록 검색 서비스 제공
- 주간/일간 랭킹을 통한 인기 공연 정보 제공
- 일정에 따른 공연 정보 제공
- Kakao Api를 활용한 로그인 시스템
- KakaoPay Api를 활용한 결제 시스템
- 관리자 페이지를 통한 매출 확인
- 관리자가 게시물 관리 및 광고 등록 가능
- 관리자 페이지를 통한 회원관리(블랙리스트)
- 크롤링을 통한 웹 예매 정보 파싱
